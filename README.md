<h1>Meditech Web</h1>
Es un sistema de gestión medica desarrollado como aplicación web, diseñado para optimizar la administracion de procesos clínicos en una institución de salud.
El sistema permite centralizar y organizar la información relacionada con citas medicas, historial clinico, gestion de usuarios, etc. facilitando el control y seguimiento de la gestion medica.

<h2>Administracion de usuarios</h2>
<img width="1441" height="825" alt="image" src="https://github.com/user-attachments/assets/4c9c87a8-6849-416f-b370-3a2215962b04" />
Permite administrar las personas que utilizan el sistema Meditech. A través de este espacio se pueden registrar nuevos usuarios, actualizar su información y controlar el acceso a la plataforma. Cada usuario puede tener un rol diferente dentro del sistema, dependiendo de las funciones que deba realizar.

<h2>Gestion de citas medicas</h2>
<img width="1445" height="820" alt="image" src="https://github.com/user-attachments/assets/d2581c61-3486-4aec-a1cc-96a710f732bf" />

Organiza y gestiona las consultas entre pacientes y profesionales de la salud. Desde aquí se pueden programar nuevas citas o cancelarlas, facilitando la planificación de la atención médica y evitando confusiones en la agenda.

<h2>Administración de historial clínico</h2>
<img width="1442" height="790" alt="image" src="https://github.com/user-attachments/assets/5d98ac77-83df-40c9-a666-0a28d41998a6" />
Almacena la información médica de los pacientes. En él se registran los diagnósticos, observaciones y tratamientos indicados por los profesionales de la salud durante las consultas. De esta manera se puede llevar un seguimiento ordenado del estado de salud de cada paciente.
